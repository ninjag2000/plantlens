## 9. Краткий чек-лист ASO

- [ ] Name ≤ 30 символов, с брендом и пользой
- [ ] Subtitle ≤ 30 символов, ключевые слова
- [ ] Keywords до 100 символов, без пробелов, без повторов из названия/подзаголовка
- [ ] Description: сильный первый абзац (2–3 строки)
- [ ] Promotional text обновлять под акции/сезон
- [ ] Категория: Reference или Lifestyle + вторичная
- [ ] Скриншоты: 3–7 штук с подписями по ключевым фичам
- [ ] Иконка 1024×1024 узнаваема и читаема в маленьком размере
- [ ] Локализовать название/подзаголовок/описание для основных языков (EN, RU, DE, FR, ES)
